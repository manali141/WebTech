# WebTech
A responsive web application built using modern Web Technologies including HTML, CSS, and Font Awesome. This project showcases a clean UI design with a navigation bar, responsive card layout, and a footer with contact information.

#
Features
- Responsive Design
- Navigation Bar
- Card based gallery layout using Grid
- Footer
    - Contact Info
    -Contact Form
    - Social Media
- Using Fontawesome Icon
